Sobre a ideia e apresentação

Coleta dos frutos




A idéia do projeto se baseia em, a pessoa tirar uma foto da árvore, subir para o site e com os dados do gps do celular, fazer o registro de localização da mesma.

Com o tempo mais pessoas passarão no local e confirmando que sim ha uma árvore ali, será gerado um marcador de período que aquela árvore fornecerá frutos.
Assim tendo em tempo real esses dados, podemos indicar onde pessoas poderão colher esses frutos.

Frutos coletados


Com os frutos coletados, será gerada uma (planiha).

Com o tempo, teremos uma base de dados onde podemos prever a próxima vez que a árvore dará frutos



Primeira coleta repasse/repasses


No começo do projeto todos terão acesso aos locais de coleta ao mesmo tempo.

Apôs um tempo será divido por cadastro e futuramente por subclasses.

A primeira tentativa é de dar preferência para pessoas de baixa renda.

Possivel união com o sistema CadÙnico para localizar e indentificar essas pessoas.

Após isso será liberado o ponto para todos.



Futuras ideias



Levantar dados para empresas focadas em sustentabilidade

Levantar dados para prefeituras

Usar os dados para indicar locais sem pontos de coleta

Criar uma forma onde pessoas que tem árvores em casa possam doar seus frutos

Gerar pontos de distribuição de frutos (marcar locais, árvores, postes ou até criar um sistema de caixas coletoras)

Incentivar e indicar as melhores árvores para plantiu, com base no local de referência do usuário
